# Titanic Data Analysis and Visualization

## 📌 Project Overview

This project performs exploratory data analysis and visualization on the Titanic dataset using Python.

The objective is to analyze passenger information, identify patterns, handle missing values, and visualize relationships between different features.

## 📊 Dataset

The dataset contains information about 891 Titanic passengers and 15 features, including:

- Passenger class
- Gender
- Age
- Number of siblings/spouses
- Number of parents/children
- Fare
- Survival status
- Embarkation details

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 🔍 Analysis Performed

The following analysis was performed:

- Loaded the Titanic dataset using Pandas
- Inspected the first few rows
- Analyzed dataset information and data types
- Generated statistical summaries
- Checked for missing values
- Calculated the average passenger age
- Analyzed passenger class distribution
- Studied the relationship between age and fare
- Created a correlation heatmap
- Analyzed survival counts
- Compared male and female passenger counts

## 📈 Visualizations

The project includes:

1. Passenger Class Distribution Bar Chart
2. Age vs Fare Scatter Plot
3. Correlation Heatmap
4. Passenger Survival Count Chart
5. Male vs Female Passenger Distribution Chart

## 📌 Key Observations

- The dataset contains 891 passenger records and 15 columns.
- The average passenger age is approximately 29.7 years.
- Most passengers travelled in Third Class.
- Missing values are present mainly in the Age and Deck columns.
- The number of passengers who did not survive was higher than the number who survived.
- The dataset contains more male passengers than female passengers.
- The scatter plot helps visualize the relationship between passenger age and fare.
- The correlation heatmap shows relationships between numerical features.
