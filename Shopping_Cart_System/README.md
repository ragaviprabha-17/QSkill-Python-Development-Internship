# Shopping Cart System

## 📌 Project Overview

The Shopping Cart System is a Python-based console application that simulates a basic e-commerce shopping experience.

Users can view available products, add multiple products to a shopping cart, select quantities, calculate the total bill, apply automatic discounts, and generate a final shopping receipt.

This project was developed as part of the QSkill Python Development Internship.

## 🚀 Features

- Product catalogue with multiple products
- Add multiple products to the shopping cart
- Quantity selection
- Cart management
- Automatic total price calculation
- Discount system based on purchase amount
- Final checkout receipt
- Input validation and error handling

## 🛠️ Technologies Used

- Python
- Google Colab

## ⚙️ Discount Rules

| Total Purchase Amount | Discount |
|---|---:|
| ₹50,000 and above | 20% |
| ₹25,000 – ₹49,999 | 10% |
| ₹10,000 – ₹24,999 | 5% |
| Below ₹10,000 | No discount |

## 🔄 How the System Works

1. The product catalogue is displayed.
2. The user selects products.
3. The user enters the required quantity.
4. Products are added to the shopping cart.
5. The system calculates the total price.
6. An automatic discount is applied.
7. A final shopping receipt is generated.
