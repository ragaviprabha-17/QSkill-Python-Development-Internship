# Matrix Operations Tool

## 📌 Project Overview

The Matrix Operations Tool is a Python-based application developed using NumPy. It allows users to enter custom matrices and perform various mathematical operations through an interactive menu.

This project was developed as part of the QSkill Python Development Internship.

## 🚀 Features

The tool supports the following matrix operations:

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Element-wise Multiplication
- Matrix Transpose
- Determinant Calculation
- Matrix Inverse

## 🛠️ Technologies Used

- Python
- NumPy
- Google Colab

## ⚙️ How It Works

1. The user enters the number of rows and columns.
2. The user enters the values for Matrix A.
3. The user enters the values for Matrix B.
4. The program displays an interactive menu.
5. The user selects the required matrix operation.
6. The program displays the result.

## 🧮 Supported Operations

### Addition

Adds corresponding elements of two matrices.

### Subtraction

Subtracts corresponding elements of Matrix B from Matrix A.

### Matrix Multiplication

Performs standard matrix multiplication using NumPy.

### Element-wise Multiplication

Multiplies corresponding elements of two matrices.

### Transpose

Converts rows into columns.

### Determinant

Calculates the determinant of a square matrix.

### Inverse

Calculates the inverse of a square matrix when the determinant is non-zero.

## 🛡️ Error Handling

The program includes:

- Invalid menu choice handling
- Invalid input handling
- Square matrix validation for determinant and inverse
- Detection of matrices without an inverse
