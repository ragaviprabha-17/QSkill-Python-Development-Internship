# 💰 Expense Tracker

A Python-based expense management application developed during the QSkill Python Development Internship.

This project allows users to record, manage, analyze, visualize, and save their daily expenses.

---

## 🚀 Features

- Add multiple expenses
- View all recorded expenses
- Calculate total expenses
- Perform category-wise expense analysis
- Search expenses by category
- Visualize expenses using bar charts
- Visualize expense distribution using pie charts
- Save expense data to a CSV file
- Load saved expense data from CSV
- Interactive menu-based system
- Input validation for expense amounts

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Google Colab

---

## 📊 Visualizations

The project includes:

### Bar Chart
Displays the total amount spent in each category.

### Pie Chart
Shows the percentage distribution of expenses across different categories.
